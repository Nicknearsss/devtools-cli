# Changelog

## [1.0.0] - Initial Release
- Initial feature set: git-autopush, debug-suite, sys-info, repo-cleaner, branch-sync, error-trace
- Installer and npm packaging support

## [1.0.1] - More Advanced Tools Added (current)
- Added **smart-git-clone**: clones repos, detects packages, auto-installs dependencies, offers to run dev server, supports shallow clone, fetches GitHub stars/forks/issues/languages
- Added **code-check**: analyzes code quality, detects warnings, errors, and suspicious patterns
- Added **env-creator**: interactive `.env` file generator
- Added **postinstall jq installer**: prompts users to install `jq` if missing during NPM install
